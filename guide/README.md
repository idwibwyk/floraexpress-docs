## Краткий обзор интерфейса

Ниже представлена таблица с описанием основных экранов приложения.

| Экран приложения | Описание |
|------------------|----------|
| **Каталог** | Просмотр всех доступных букетов и композиций. Есть фильтры по цене и поводу. |
| **Избранное** | Сохраненные вами товары для быстрого доступа. |
| **Корзина** | Просмотр выбранных товаров, применение промокодов и оформление заказа. |
| **Профиль** | Управление вашими данными, адресами и просмотр истории заказов. |
